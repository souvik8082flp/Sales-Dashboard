# DashboardSales Power BI Template

## Overview

This repository contains the **DashboardSales.pbit** Power BI template.\
It is designed to provide an interactive and visually appealing
dashboard for analyzing sales data across different metrics and time
periods.

## Features

-   **Sales Overview**: Track total sales, revenue trends, and key
    performance indicators.
-   **Category-wise Performance**: Analyze sales distribution across
    product categories.
-   **Region-based Insights**: View sales performance by geographical
    regions.
-   **Timeâ€‘based Analysis**: Compare sales across months, quarters, and
    years.
-   **Dynamic Filtering**: Use slicers to customize the displayed data.

## Requirements

-   **Microsoft Power BI Desktop**
-   A compatible dataset matching the template structure\
    (Power BI will prompt for data source parameters on opening the
    `.pbit` file.)

## How to Use

1.  Download the file `DashboardSales.pbit`.
2.  Open it with **Power BI Desktop**.
3.  When prompted, connect your data source and update the parameters.
4.  Refresh the data.
5.  Customize visuals as needed.
